# bareunmal

Vue 3 + Pinia + Vite + TypeScript 기반의 기본 스캐폴드입니다.

## 시작하기
```bash
npm install
npm run dev
```

## 빌드
```bash
npm run build
```

## 구조
```text
src/
  app/
  pages/
  widgets/
  features/
  entities/
  shared/
  core/
    domain/
    application/
  adapters/
    vue/
      stores/
      composables/
```
